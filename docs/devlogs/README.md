# Devlogs

這裡放實作導向的開發日誌。和 phase 記錄不同，devlog 允許更細地解釋：

- 為什麼需要這一段工作
- 每個檔案是做什麼的
- 哪些邊界是刻意凍結的
- 如何驗證
- 下一步要接什麼
